# 微信公众号文章摘要生成器

## 功能描述
智能生成吸引人的文章摘要，提高文章在搜索结果、转发分享中的点击率，快速传达文章核心价值。

## 输入参数
- **文章全文**: 完整的文章内容
- **核心主题**: 文章主要讨论的话题
- **目标受众**: 读者群体特征
- **文章类型**: 干货/故事/评测/观点等
- **摘要长度**: 100字/150字/200字

## 摘要类型模板

### 1. 干货类摘要
**特点**：突出价值感和实用性
**公式**：[痛点] + [解决方案] + [预期收获] + [行动号召]

示例：
```
还在为每天写不出内容而焦虑吗？本文分享3个被验证有效的AI写作技巧，让效率提升300%，零基础也能快速上手。包含具体工具推荐和实操案例，建议收藏反复查看。

#点击率预估：85%
```

### 2. 故事类摘要
**特点**：营造情感共鸣，激发好奇心
**公式**：[冲突] + [转折] + [成长] + [启示]

示例：
```
从月薪3000到年入百万，他只做对了一件事。3年前还在工厂打工的李明，因为偶然接触到AI写作，人生彻底改变。这个真实故事告诉你，普通人如何抓住AI风口完成逆袭。

#点击率预估：78%
```

### 3. 评测类摘要
**特点**：强调专业性和客观性
**公式**：[评测对象] + [核心发现] + [优缺点] + [推荐建议]

示例：
```
深度测评5款主流AI写作工具，从功能、价格、效果等多维度对比。实测发现，工具A最适合新手，工具B专业度最高，工具C性价比最优。附详细对比表格和使用建议。

#点击率预估：82%
```

### 4. 观点类摘要
**特点**：引发思考和讨论
**公式**：[现象] + [独特观点] + [论证] + [互动]

示例：
```
AI写作真的会取代人类作者吗？作为一名从业10年的编辑，我认为AI只是工具而非替代。本文深度分析AI写作的3大局限和人类创作者不可替代的价值。你怎么看？

#点击率预估：75%
```

### 5. 新闻类摘要
**特点**：突出时效性和重要性
**公式**：[事件] + [影响] + [细节] + [趋势]

示例：
```
重磅！微信推出AI写作助手功能，千万创作者迎来变革。实测发现，新功能可自动生成文章大纲、优化标题，效率提升显著。解读功能详情和使用技巧，抢占先机必看。

#点击率预估：88%
```

## 摘要生成流程

### 第一步：内容分析
```
1. 提取核心观点
   - 识别文章主要论点
   - 找出关键数据和案例
   - 总结价值主张

2. 分析文章结构
   - 确定内容类型
   - 识别故事线或逻辑线
   - 定位高潮部分

3. 挖掘亮点元素
   - 数字、数据、百分比
   - 冲突、转折、成长
   - 秘密、技巧、方法
```

### 第二步：受众匹配
```
用户画像：
- 年龄段：25-35岁
- 职业：内容创作者
- 痛点：效率低、缺灵感
- 需求：实用技巧、工具推荐

匹配策略：
- 使用"效率"、"提升"等词
- 强调"零基础也能用"
- 突出"实测"、"验证"
```

### 第三步：优化文案
```
吸引力元素：
1. 数字具体化
   - "很大提升" → "提升300%"
   - "很多方法" → "3个方法"

2. 对比强化
   - "之前vs之后"
   - "普通vs专业"
   - "手动vs自动"

3. 情绪调动
   - 焦虑 → 解脱
   - 困惑 → 清晰
   - 怀疑 → 确信

4. 权威背书
   - "专家建议"
   - "实测验证"
   - "10年经验"
```

### 第四步：SEO优化
```
关键词布局：
- 主关键词：AI写作
- 相关词：效率、工具、技巧
- 长尾词：AI写作工具推荐

优化技巧：
- 标题词前置
- 自然融入不堆砌
- 100字内包含2-3个关键词
```

## 不同场景的摘要策略

### 1. 搜索结果展示
- **长度**：100-120字
- **重点**：包含核心关键词，突出价值
- **技巧**：前置重要信息，使用数字

### 2. 朋友圈分享
- **长度**：150字以内
- **重点**：引发好奇心，促进点击
- **技巧**：使用疑问句，制造悬念

### 3. 转发给朋友
- **长度**：80-100字
- **重点**：突出对朋友的价值
- **技巧**：个性化推荐，说明适用人群

### 4. 微信群分享
- **长度**：100-150字
- **重点**：引发讨论，适合群主题
- **技巧**：使用@提及，设置互动点

## A/B测试框架

### 测试变量
1. 开头方式（提问/陈述/感叹）
2. 数据呈现（具体/概括）
3. 情绪倾向（理性/感性）
4. 行动号召（强/弱）

### 评估指标
- **点击率（CTR）**
- **阅读完成率**
- **分享转化率**
- **停留时长**

### 优化迭代
```
版本A：点击率5.2%
版本B：点击率7.8%
优化幅度：+50%

胜出特征：提问式开头+具体数据
```

## 常见错误规避

### 1. 标题党式摘要
```
错误：震惊！AI写作竟让年薪百万！
正确：实测：用AI写作3个月，副业收入超主业
```

### 2. 信息过载
```
错误：本文将详细介绍AI写作的发展历史、技术原理、应用场景、工具推荐、使用技巧...
正确：5分钟了解AI写作的核心技巧和工具推荐
```

### 3. 无关承诺
```
错误：读完本文，月入过万不是梦！
正确：掌握这些技巧，写作效率提升看得见
```
