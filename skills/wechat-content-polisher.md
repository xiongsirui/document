# 微信公众号文章润色优化器

## 功能描述
对文章内容进行全面润色和优化，提升文章质量、可读性和传播力，确保符合微信公众号平台的特色。

## 输入参数
- **原文内容**: 需要优化的文章内容
- **优化目标**: 提升可读性/增加传播性/优化SEO/增强情感共鸣
- **文章类型**: 原创/转载/干货/故事等
- **目标受众**: 读者群体特征

## 润色优化维度

### 1. 语言表达优化
- **用词精准化**
  - 替换模糊词汇为具体表达
  - 去除口语化表达
  - 统一专业术语使用
  - 避免重复用词

- **句式多样化**
  - 长短句搭配
  - 主动被动语态交替
  - 排比句增强气势
  - 设问句引发思考

- **语气调整**
  - 根据内容调整正式/轻松程度
  - 保持一致性
  - 适当使用感叹号增强情感
  - 避免过于绝对的表达

### 2. 逻辑结构优化
- **段落重组**
  - 确保每段一个中心思想
  - 调整段落长度（3-5行为佳）
  - 添加过渡句增强连贯
  - 重要段落前置

- **信息层次**
  - 突出核心观点
  - 删除冗余信息
  - 补充必要细节
  - 确保论证完整

- **开头结尾强化**
  - 开头3秒吸引力优化
  - 结尾行动号召加强
  - 金句提炼和突出
  - 情感升华设计

### 3. 可读性提升
- **排版建议**
  - 添加小标题分割内容
  - 使用项目符号列表
  - 关键内容加粗强调
  - 段落间空行分隔

- **阅读节奏**
  - 控制句子长度（15-20字）
  - 避免长段落（超过8行）
  - 适当留白
  - 视觉重点突出

### 4. 情感共鸣强化
- **情感词植入**
  - 添加感同身受的表达
  - 使用情绪化词汇
  - 设计情感转折点
  - 营造代入感

- **故事化包装**
  - 数据用案例说明
  - 抽象概念具象化
  - 增加个人经历分享
  - 设置冲突和解决

### 5. 互动引导设计
- **提问植入**
  - 在关键节点设置问题
  - 引导读者思考
  - 促进评论区互动
  - 增加参与感

- **分享引导**
  - 设计转发理由
  - 提供分享价值点
  - 植入"转发给需要的人"等提示
  - 制作金句卡片

### 6. SEO关键词优化
- **关键词布局**
  - 标题包含主关键词
  - 开头100字自然融入
  - 小标题使用相关词
  - 结尾总结提及

- **长尾词优化**
  - 自然融入相关长尾词
  - 使用近义词避免重复
  - 添加地域性修饰（如适用）
  - 结合热点词汇

- [ ] 不违反广告法

### 传播性检查
- [ ] 有记忆点

## 优化建议类型

### 1. 必改项（红色标记）
- 错别字和语病
- 事实性错误
- 平台违规内容
- 严重逻辑问题

### 2. 建议修改（黄色标记）
- 表达不够精准
- 句子过于冗长
- 逻辑不够清晰
- 可读性较差

### 3. 可优化（蓝色标记）
- 可以更生动
- 可以更精炼
- 可以更有感染力
- 可以更利于传播

## 优化报告格式

```
文章润色优化报告

总体评分：85/100

主要亮点：
✓ 开头吸引人
✓ 结构清晰
✓ 内容实用

优化建议：

1. 语言表达（需要修改）
   - 第3段第2行："非常"替换为"极其"
   - 第5段：建议拆分为两个段落

2. 逻辑结构（建议优化）
   - 建议调整第2部分和第3部分顺序
   - 补充过渡句：基于以上分析...

3. 互动引导（可以增强）
   - 在第4部分末尾添加提问
   - 结尾添加分享引导

4. 关键词优化
   - 建议在标题加入"AI写作"
   - 正文3处可优化关键词密度

优化后预览：
[提供关键段落优化后的示例]

终审建议：
✓ 适合发布
```

## 使用示例
```
输入原文：
"今天我要分享一个工具，这个工具很好用，可以帮助你写文章。我觉得大家都应该试试..."

优化后：
"作为一名每天需要输出5000字的写作者，我终于找到了改变游戏规则的AI写作工具。它不仅让我的创作效率提升了300%，更重要的是，让我重新找回了写作的乐趣..."

优化说明：
1. 增加具体身份和数字，增强可信度
2. 使用"游戏规则"、"乐趣"等词汇增加感染力
3. 通过对比突出工具价值
```

## 进阶功能
- **多种风格切换**：正式/轻松/幽默/专业
- **阅读时间预估**：根据优化后内容计算
- **AI配图建议**：根据内容推荐配图方向
- **发布时间建议**：根据受众活跃时间推荐
