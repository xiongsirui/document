# 微信公众号标题生成器

## 功能描述
根据文章内容智能生成吸引眼球的微信公众号标题，提供多种风格选择，提高文章点击率。

## 输入参数
- **核心主题**: 文章主要讨论的内容
- **关键词**: 需要包含的关键词（最多3个）
- **文章类型**: 指南/评测/资讯/故事/观点等
- **目标受众**: 主要读者群体
- **标题风格**: 悬念/数字/疑问/情感/痛点等

## 标题生成模式

### 1. 数字型标题
- 提高可信度和阅读欲望
- 适合盘点、攻略、教程类文章
- 示例：
  - "3个技巧让你的阅读量翻倍"
  - "2024年AI写作的5大趋势"
  - "月薪3千到3万，我只用了这7个方法"

### 2. 疑问型标题
- 引发好奇心，促使点击
- 适合解答、揭秘类内容
- 示例：
  - "为什么你的文章没人看？"
  - "AI写作真的能取代人类吗？"
  - "如何写出10万+爆文？"

### 3. 悬念型标题
- 制造神秘感，激发探索欲
- 适合故事、案例、揭秘内容
- 示例：
  - "我只用了一招，就让老板主动给我加薪..."
  - "这个AI功能，99%的人都没发现"
  - "月入5万的自媒体人，都懂这个秘密"

### 4. 痛点型标题
- 直击用户痛点，提供解决方案
- 适合教程、指南、解决方案类
- 示例：
  - "还在为选题发愁？这个工具帮你搞定"
  - "写作没灵感？试试这4个方法"
  - "别再用错误的方式写公众号了"

### 5. 情感型标题
- 调动情绪共鸣，增强传播性
- 适合观点、故事、感悟类内容
- 示例：
  - "写了100篇文章后，我终于懂了..."
  - "感谢那些质疑我的人，让我学会了坚持"
  - "作为写作者，最痛苦的莫过于..."

### 6. 干货型标题
- 突出实用价值，吸引目标用户
- 适合干货、技能、知识分享
- 示例：
  - "超强干货：AI写作完整指南（附工具清单）"
  - "手把手教你搭建个人写作系统"
  - "内部流出：大号编辑的10个写作秘诀"

### 7. 对比型标题
- 通过对比突出价值
- 适合评测、对比、经验分享
- 示例：
  - "用AI写作vs人工写作，差距竟然这么大"
  - "同样写公众号，为什么他赚得比你多？"
  - "月薪5000和5万的自媒体人，差的不只是写作能力"

## 优化建议

### 标题长度控制
- 最佳长度：15-25个字
- 手机端完整显示：不超过20字
- 关键信息前置

### 关键词布局
- 核心关键词放在前半部分
- 避免关键词堆砌
- 自然融入标题

### 情绪词运用
- 使用"竟然"、"终于"、"居然"等情绪词
- 加入"必看"、"重磅"、"独家"等强调词
- 慎用"震惊"、"惊呆"等过度夸张词

### 符号运用
- 适当使用【】、《》、「」等符号
- 数字用阿拉伯数字更醒目
- 避免过度使用表情符号

## 标题评分标准
1. **吸引力**（30%）：是否引发好奇心
2. **相关度**（25%）：是否准确反映内容
3. **搜索性**（20%）：是否利于搜索
4. **传播性**（15%）：是否利于转发
5. **原创性**（10%）：是否独特新颖

## A/B测试框架
- 提供2-3个备选标题
- 建议测试时长：24-48小时
- 监控指标：点击率、阅读完成率、分享率

## 禁用词提醒
- 避免使用法律法规禁用词
- 谨慎使用医疗健康类词汇
- 避免夸大宣传用语
