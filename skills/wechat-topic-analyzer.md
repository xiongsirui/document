# 微信公众号选题分析器

## 功能描述
分析选题的潜在表现、竞争程度和可行性，为微信公众号内容创作提供数据支持。

## 输入参数
- **主题**: 文章选题或核心关键词
- **目标受众**: 目标读者群体描述
- **内容类型**: 文章类型（资讯、教程、评测、故事等）

## 输出内容

### 1. 热度分析
- 微信指数趋势
- 搜索热度变化
- 季节性波动特征

### 2. 竞争评估
- 同类内容数量
- 头条号表现
- 差异化机会

### 3. 受众匹配度
- 话题与受众匹配度
- 潜在阅读兴趣
- 传播潜力评估

### 4. 内容建议
- 推荐切入角度
- 亮点提炼方向
- 避免的雷区

### 5. SEO优化提示
- 核心关键词
- 长尾关键词组合
- 标题优化建议

### 6. 发布时机
- 最佳发布时间
- 热点结合建议
- 跟进策略

## 使用示例
```
输入：
主题：AI写作工具
目标受众：内容创作者、自媒体从业者
内容类型：评测教程

输出：
热度分析：近期搜索热度上升30%，属于上升趋势
竞争评估：中等竞争，头部账号已占位，但细分市场有机会
受众匹配度：高度匹配（90%）
内容建议：对比评测+实操教程+变现思路
SEO优化：主关键词"AI写作"，辅以"AI写稿"、"智能写作"等
```

## 实现要点
- 需要接入微信指数API
- 建立热点话题数据库
- 结合历史数据分析
- 提供可执行的建议

## 评分维度
1. 热度指数（0-100分）
2. 竞争激烈程度（低/中/高）
3. 传播潜力（0-100分）
4. 推荐指数（综合评分）
