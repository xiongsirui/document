# 微信公众号SEO关键词优化器

## 功能描述
针对微信公众号平台的搜索特性，优化文章关键词布局，提高文章在微信搜索中的曝光率和排名。

## 输入参数
- **文章主题**: 核心内容主题
- **目标关键词**: 主要优化关键词
- **文章内容**: 完整文章内容
- **目标人群**: 搜索受众画像

## 微信SEO核心要素

### 1. 关键词研究与选择
- **主关键词**
  - 搜索热度高（微信指数>500）
  - 与内容高度相关
  - 竞争度适中
  - 转化潜力大

- **长尾关键词**
  - 搜索意图明确
  - 竞争度低
  - 转化率高
  - 组合灵活

- **相关关键词**
  - 语义相关
  - 用户搜索关联
  - 行业热词
  - 时事热点

### 2. 关键词布局策略
- **标题优化**
  - 主关键词前置（前5个字）
  - 控制标题长度（15-25字）
  - 避免关键词堆砌
  - 自然融入表达

- **摘要优化**
  - 100-150字摘要
  - 包含2-3个关键词
  - 突出核心价值
  - 引导点击

- **正文优化**
  - 开头100字内出现主关键词
  - 关键词密度2%-8%
  - 小标题使用关键词
  - 自然分布避免堆砌

- **图片优化**
  - 图片名称包含关键词
  - ALT描述文字优化
  - 图文相关性高
  - 压缩后上传

### 3. 微信搜索算法特点
- **时效性权重**
  - 新文章有优先展示
  - 热点话题加权
  - 更新频率影响
  - 发布时间重要

- **原创性评估**
  - 原创内容加权
  - 抄袭内容降权
  - 引用需标注
  - 改写需深度

- **用户行为反馈**
  - 阅读完成率
  - 点赞收藏数
  - 分享转发量
  - 评论互动量

- **账号权重**
  - 账号认证状态
  - 发布频率
  - 粉丝质量
  - 历史表现

## 关键词密度控制

### 理想密度范围
- **主关键词**：2%-3%
- **长尾词**：1%-2%
- **相关词**：3%-5%
- **总关键词密度**：不超过8%

### 计算方法
```
密度 = (关键词出现次数 × 关键词字数) ÷ 总字数 × 100%

示例：
文章总字数：2000字
主关键词"AI写作"（4字）出现8次
密度 = (8 × 4) ÷ 2000 × 100% = 1.6%
```

### 优化建议
- 避免刻意堆砌
- 保持语句通顺
- 重点位置突出
- 自然分布最佳

## 常见错误规避

### 1. 关键词堆砌
```
错误案例：
"AI写作、AI写作工具、最好AI写作、AI写作推荐"

正确做法：
"深度测评：2024年最值得使用的AI写作工具"
```

### 2. 内容不相关
```
错误案例：
标题写"AI写作"，内容讲"新媒体运营"

正确做法：
确保标题与内容高度匹配
```

### 3. 过度优化
```
错误案例：
每段都强行植入关键词

正确做法：
自然融入，用户体验优先
```
