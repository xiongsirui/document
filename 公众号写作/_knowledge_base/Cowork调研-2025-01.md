# Cowork 调研资料

**调研时间**：2025-01-13
**产品**：Claude Cowork
**状态**：研究预览阶段（Research Preview）

---

## 一、产品基本信息

### 1.1 产品定义
- **全称**：Claude Cowork
- **发布时间**：2025年1月
- **产品定位**：Claude Code 的"白领版"，面向非技术用户
- **所属平台**：Claude macOS 桌面应用

### 1.2 核心理念
将 Claude Code 的 AI 代理能力从编程领域扩展到日常办公任务，让没有编程背景的用户也能享受到 AI 代理带来的便利。

---

## 二、核心功能

### 2.1 文件夹访问权限
- 可以授予 Claude 访问计算机上特定文件夹的权限
- Claude 可以读取、编辑或创建该文件夹中的文件

### 2.2 智能文件管理
- 像整理文件夹这样的基础任务可以自主完成
- 能够组织、创建和编辑文档
- 一键清空桌面

### 2.3 非技术任务自动化
- 让非程序员用户也能像开发者使用 Claude Code 一样流畅地完成工作
- 适用于日常办公和文档处理任务

### 2.4 任务自主执行
- 设置任务后，Claude 会制定计划并稳步完成
- 向用户实时通报进展情况
- 类似真正的助手，能够自己规划步骤、一步步执行

---

## 三、产品特点

### 3.1 更强的自主性
- 设置任务后，Claude 会制定计划并稳步完成
- 向用户实时通报进展情况
- 类似真正的助手，能够自己规划步骤、一步步执行

### 3.2 极低的使用门槛
- 开箱即用，上手简单
- 无需持续提供背景信息
- 无需手动转换输出格式
- 本质上是把强大的代码执行能力包装在不吓人的界面里

### 3.3 多任务并行处理
- 可同时排定多项任务
- 支持系统平行处理
