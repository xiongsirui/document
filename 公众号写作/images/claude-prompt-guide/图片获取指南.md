3. **03-information-overload.jpg** - 过度补偿：信息过载
   - 描述：过多的文字或信息让人困惑

4. **04-aha-moment.jpg** - 顿悟时刻：灯泡亮起
   - 描述：灵光一闪的瞬间，灯泡或闪光效果
   - 推荐搜索词：aha moment, light bulb idea, eureka

   - 描述：两个人完美配合，默契十足的场景
   - 推荐搜索词：perfect teamwork, understanding, collaboration success
   - 推荐来源：Unsplash, Pexels

## 免费图库推荐

1. **Unsplash** (unsplash.com)
   - 高质量摄影图片
   - 免费商用，无需署名

2. **Pexels** (pexels.com)
   - 图片和视频
   - 免费商用，无需署名

3. **Freepik** (freepik.com)
   - 大量矢量图和插画
   - 免费版需署名，付费版可免署名

4. **Vecteezy** (vecteezy.com)
   - 免费矢量图资源
   - 需注意部分需要署名

## 图片处理建议

1. **尺寸统一**：建议宽度1200px，高度根据图片内容调整
2. **风格一致**：尽量选择相似的色调和风格
3. **格式选择**：JPG格式，质量85-90%
4. **文件大小**：单张图片控制在500KB以内

## 注意事项

- 确保所有图片都有商用许可
- 保存原始来源链接，以备查证
- 如需要，在文章末尾注明图片来源
