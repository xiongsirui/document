## OpenClaw 是什么

### 传统 AI 助手的局限

你现在用的 AI 工具，大概率是这样：

| 类型 | 能做什么 | 做不到 |
|------|----------|---------|
| ChatGPT / Claude | 聊天、写代码、分析文档 | 操作你的电脑、自动执行 |
| Claude Code / Cursor | 写代码、重构项目 | 非 IDE 场景的自动化 |

**核心问题**：AI 被困在"对话框"里，出不来。

### OpenClaw 的突破

用官方的话说：OpenClaw 是 **"通用的 Agent 应用程序和技能框架"**。

**用人话说**：
1. **能操作你的电脑**：打开浏览器、登录网站、点击按钮
2. **技能可扩展**：像手机装 APP 一样，想装什么技能就装什么
3. **跨平台**：Windows、macOS、Linux 都支持
4. **AI 模型自由**：可以用 Claude、GPT-4、本地模型...

**对比一下**：

| Claude（传统） | OpenClaw + Claude |
|--------------|------------------|
| 聊天问答 | 聊天 + 自动执行操作 |
| 你问一句它答一句 | 它可以主动完成任务 |
| 无法操作外部应用 | 能控制浏览器、系统应用 |

## 实战场景演示

### 场景 1：自动监控网页并发送通知

**需求**：监控某个网页（比如产品价格、招聘信息），有变化了通知我。

**传统方式**：手动打开网页、F5 刷新、肉眼对比

**OpenClaw 方式**：安装"网页监控"技能，配置监控 URL + 触发条件，AI 每 10 分钟自动检查一次，变化了发通知到手机。

---

### 场景 2：批量处理 Excel 报表

**需求**：每个月收到一堆 Excel，需要汇总、分析、生成报告。

**OpenClaw 方式**：把一堆 Excel 扔给 AI，说："汇总这个月的销售数据，生成报告"。AI 自动打开所有文件、提取数据、汇总计算、生成图表、输出报告。

**省下的**：原来 2 小时工作，10 分钟搞定。

---

### 场景 3：社交媒体内容自动化

## 从零搭建教程

### 步骤 1：安装 OpenClaw

**方法 A：一键安装（推荐新手）**

访问 https://www.openclaw.com/，下载安装包，运行安装程序，选择 AI 模型（推荐 Claude 或本地模型）。

**方法 B：Docker 部署（推荐有技术基础的人）**

```bash
docker run -d openclaw/app
```

---

### 步骤 2：配置 AI 模型

| 选项 | 优点 | 缺点 |
|------|------|------|
| Claude API | 推理质量最好，能力强 | 按 token 计费 |
| GPT-4 API | 综合能力强 | 按 token 计费 |
| 本地模型 | 完全免费 | 对电脑配置要求高 |

**推荐**：本地模型（Ollama + LM Studio）免费且隐私安全。

---

### 步骤 3：安装第一个技能

打开 ClawHub（技能市场），搜索"web monitor"或"网页监控"，点击"安装"，配置参数（监控 URL、频率、通知方式），启动。就完成了。
