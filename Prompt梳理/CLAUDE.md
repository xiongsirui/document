# Prompt梳理工作区

## 功能说明
管理和优化各类写作提示词（Prompt）。

## 工作内容
1. **Prompt收集**
   - 收集高效写作模板
   - 整理行业特定提示词

2. **分类管理**
   - 按场景分类（写作、编程、分析等）
   - 标注适用范围和效果

3. **优化测试**
   - A/B测试不同版本
   - 记录最佳实践

4. **版本控制**
   - Git管理Prompt版本
   - 记录迭代历史

## Prompt模板库
- 开篇模板
- 转场句式
- 结尾模板
- 行业专用术语库
- 风格转换指南

## 使用方法
1. 搜索合适的模板
2. 根据需求调整参数
3. 一键应用到写作任务
