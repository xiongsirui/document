# GLM内容分析器

专业的文本分析工具，提供多维度内容洞察。

## 分析功能

### 写作风格分析
分析文本的写作风格、语言特征和表达习惯。

**使用示例：**
```
请用glm_analyzer分析这段文字的写作风格："在当今这个快速发展的时代..."
```

### 情感分析
识别文本的情感倾向和情感强度。

**使用示例：**
```
请用glm_analyzer分析这段评论的情感是积极还是消极："这个产品真的很棒，我非常喜欢！"
```

### 关键词提取
自动提取文本的核心关键词和主题。

**使用示例：**
```
请用glm_analyzer提取这篇文章的主要关键词："文章内容..."
```

### 可读性分析
评估文本的可读性并提供改进建议。

**使用示例：**
```
请用glm_analyzer分析这段话的可读性："复杂的技术术语和长句..."
```

### 质量评估
从多个维度评估内容质量。

**使用示例：**
```
请用glm_analyzer评估这篇文章的质量，关注准确性和逻辑性："文章内容..."
```

## 应用场景

- **内容审核**：快速评估内容质量
- **SEO优化**：提取关键词，优化内容
- **用户反馈分析**：分析评论情感
- **写作改进**：识别写作风格特点

分析结果包含：
