# Prompt梳理工作区

## 核心规则

### 优化标准
- ✅ 角色清晰（你是什么）
- ✅ 任务明确（要做什么）
- ✅ 输出格式（要什么格式）
- ✅ 约束条件（不能做什么）

### 结构模板
```
# 角色
你是一个[专业角色]，具有[特质]

# 任务
你的任务是[具体任务]

# 要求
1. [具体要求1]
2. [具体要求2]
3. [具体要求3]

# 输出格式
[明确的输出格式]

# 约束
- [约束1]
- [约束2]
```

### 优化流程
1. **理解意图**：原始prompt想做什么？
2. **补充角色**：应该让AI扮演什么？
3. **明确任务**：具体要完成什么？
4. **设定格式**：希望什么输出？
5. **添加约束**：避免什么问题？

## 常见问题修复

### 问题1：角色不明确
```
❌ 原始：写一个故事
✅ 优化：你是一个资深小说家，擅长写出引人入胜的开头
```

### 问题2：任务模糊
```
❌ 原始：帮我优化代码
✅ 优化：请优化以下Python代码，重点关注性能和可读性
```

### 问题3：缺少格式
```
❌ 原始：给些建议
✅ 优化：请以列表形式给出5条具体建议，每条包含说明和示例
```

## 输出格式

```markdown
# 优化后的Prompt

## 原始版本

```

## 优化说明

## 测试建议

## Prompt类型库

- 故事创作

- 标题生成

### 分析类
- 数据分析
- 问题诊断
- 方案对比

### 技术类
- 代码生成
- 调试帮助
- 架构设计

## 最佳实践
1. **具体化**：避免抽象描述
2. **示例化**：提供好例子
3. **步骤化**：复杂任务拆解
4. **角色化**：设定专家身份
5. **格式化**：明确输出要求
