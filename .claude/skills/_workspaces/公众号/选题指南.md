# 公众号选题指南

整合选题技巧、方法和快速参考，帮助快速确定高质量选题。

---

## 三种选题方式

### 方式一：完整工作流（推荐）
```
[Workflow] 开始写作流程
主题：[你的想法]
类型：公众号
目标字数：2000-3000

# 系统会提供3-4个选题供选择
```

### 方式二：基于素材库选题
```
# 1. 先搜索你的经历
[KB] 搜索素材
关键词：[领域关键词]
类型：经历, 评价, 测试

# 2. 基于素材生成选题
[AI] 生成选题
素材：[找到的素材摘要]
要求：公众号，3000字
```

### 方式三：模板快速生成
```
使用模板：工具评测
产品：[产品名]
```

## 选题核心原则

### 1. 价值导向
- 解决读者问题
- 提供实用信息
- 引发情感共鸣

### 2. 差异化
- 找到独特角度
- 提供新见解
- 避免同质化

### 3. 可持续性
- 可系列化
- 有延展空间
- 适合长期运营

---

## 热门选题类型

### 1. 工具评测类
- **标题公式**：我用[工具名]30天后，发现…
- **特点**：真实体验，对比评测
- **素材搜索**：`[KB] 搜索 "工具名, 体验"`

### 2. 个人成长类
- **标题公式**：从[身份A]到[身份B]：我的X年转型路
- **特点**：真实经历，情感共鸣
- **素材搜索**：`[KB] 搜索 "转型, 成长"`

### 3. 行业观察类
- **标题公式**：2024年[行业]的3个关键变化
- **特点**：趋势洞察，前瞻性
- **素材搜索**：`[KB] 搜索 "趋势, 观察"`

### 4. 方法技巧类
- **标题公式**：提升[某能力]的5个实用技巧
- **特点**：实用性强，容易传播
- **素材搜索**：`[KB] 搜索 "技巧, 方法"`

### 5. 深度解读类
- **标题公式**：深度解析：[概念]的底层原理
- **特点**：专业深入，技术含量高
- **素材搜索**：`[KB] 搜索 "原理, 解析"`

## 标题优化公式

### 公式1：数字 + 结果
```
我用AI工具30天，效率提升了300%
```

### 公式2：问题 + 解决
```
代码总是出Bug？试试这个方法
```

### 公式3：对比 + 选择
```
Claude Code vs Cursor：程序员该选哪个？
```

### 公式4：故事 + 悬念
```
那个用AI写代码的程序员，后来怎么样了
```

### 公式5：主体 + 变化 + 具体成果
```
[主体] + [变化/对比/方法] + [数字/具体成果]
```

## 角度选择技巧

- **个人视角**：我/我的团队/我的经验
- **数据支撑**：统计显示/测试表明/实际效果
- **趋势敏感**：2024年/最新/即将出现

## 选题评估维度

```
[AI] 分析选题
对比以下选题：
1. 选题A：[描述]
2. 选题B：[描述]
3. 选题C：[描述]

评估标准：
- 读者兴趣度
- 内容独特性
- 写作难度
- 资源要求
- 传播潜力
```

---

## 避免的选题

- ❌ 纯理论（太枯燥）
- ❌ 泛泛而谈（没重点）
- ❌ 过时内容（无新意）
- ❌ 无实证（不靠谱）
- ❌ 标题党（内容空洞）
- ❌ 同质化（别人写过的）

---

## 好选题的特征

- ✅ 解决具体问题
- ✅ 有真实案例
- ✅ 带来新认知
- ✅ 容易引发讨论
- ✅ 标题吸引点击
- ✅ 对读者有价值

---

## 选题检查清单

选题前问自己：
- [ ] 这个话题解决了什么问题？
- [ ] 我有独特的见解或经历吗？
- [ ] 内容对目标读者有价值吗？
- [ ] 标题足够吸引人点击吗？
- [ ] 内容可持续扩展吗？
- [ ] 符合平台调性吗？
- [ ] 和其他文章有明显差异吗？
