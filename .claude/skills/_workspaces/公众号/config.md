# 公众号写作工作区

## 工作区特性

| 特性 | 状态 | 说明 |
|------|------|------|
| 需要配图 | ✅ | 至少3处配图建议 |
| SEO优化 | ✅ | 关键词布局 |
| 字数限制 | ✅ | 1500-3000字 |
| 三遍审校 | ✅ | 降AI味机制 |
| 个人素材库 | ✅ | 融入真实经历 |

---

## 内容规范

### 必须要求
- ✅ 吸引人的标题（制造悬念/使用数字/明确价值）
- ✅ 配图建议（至少3处：封面、内容、结尾）
- ✅ 字数控制：1500-3000字
- ✅ SEO关键词布局
- ✅ 结尾互动引导

### 禁忌事项
- ❌ 纯理论堆砌（没有实例）
- ❌ 没有个人观点（AI腔）
- ❌ 缺少真实案例
- ❌ 标题平淡无奇
- ❌ 编造数据和引用

---

## 文章结构模板

```markdown
# 标题（吸引点击）

## 开头（200字）
痛点/问题/悬念 → 引发共鸣

## 主体（1500-2500字）

### 观点1 + 案例
[内容 + 真实案例]

### 观点2 + 数据
[内容 + 具体数据]

### 观点3 + 方法
[内容 + 可操作步骤]

## 结尾（200字）
总结 + 行动号召

---
## 配图建议
1. 封面图：[描述]
2. 内容图1：[位置 + 描述]
3. 结尾图：[二维码位置]
```

---

## 配图要求

### 封面图
- 尺寸：推荐 900x383 或 2.35:1
- 内容：标题视觉化，吸引点击
- 风格：与文章调性一致

### 内容图
- 关键数据图表
- 产品截图（需注明来源）
- 流程示意图

### 结尾图
- 二维码引导关注
- 系列文章导航

### 图片来源优先级
1. 公共领域（Wikimedia Commons）
2. AI生成（火山引擎）
3. 免费图库（Unsplash、Pexels）
4. 官方素材（需注明来源）

## 标题公式

### 公式1：数字 + 结果
```
我用AI工具30天，效率提升了300%
```

### 公式2：问题 + 解决
```
代码总是出Bug？试试这个方法
```

### 公式3：对比 + 选择
```
Claude Code vs Cursor：程序员该选哪个？
```

### 公式4：故事 + 悬念
```
那个用AI写代码的程序员，后来怎么样了
```

## 质量检查清单

### 发布前检查
- [ ] 标题是否吸引人？
- [ ] 是否有3个以上配图建议？
- [ ] 字数是否在1500-3000字范围？
- [ ] 是否包含SEO关键词？
- [ ] 结尾是否有互动引导？
- [ ] 是否通过三遍审校？
- [ ] 数据和案例是否真实？

### AI味检查
- [ ] 删除套话（"在当今时代"、"综上所述"）
- [ ] 拆解AI句式（"不是…而是…"连续出现）
- [ ] 替换书面词（"显著提升"→具体数字）
- [ ] 口语化改写（"进行操作"→直接动词）
- [ ] 加入真实细节和个人态度
