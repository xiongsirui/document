# 公众号选题技巧与方法

本指南详细说明如何使用Skills进行公众号选题，从创意产生到最终确定。

## 🎯 选题的核心原则

### 1. 价值导向
- 解决读者问题
- 提供实用信息
- 引发情感共鸣

### 2. 差异化
- 找到独特角度
- 提供新见解
- 避免同质化

### 3. 可持续性
- 可系列化
- 有延展空间
- 适合长期运营

## 🛠️ 使用Work进行选题

### 启动完整选题流程

```bash
[Workflow] 开始写作流程
主题：[你的领域或方向]
类型：公众号
目标字数：[目标字数]
特殊要求：[其他要求]
```

### Step 3: 选题讨论环节

系统会自动进入选题讨论：

```bash
[Workflow] Step3 -选题讨论
基于Brief，提供3-4个选题方案：

选题1：AI工具对比评测
- 核心角度：工具实际使用体验对比
- 工作量：⭐⭐⭐
- 优势：有真实测试数据，可信度高
- 劣势：测试周期较长
- 是否需要真实测试：是
  大纲：
  • 开头（200字）：AI工具现状概述
  • 主体（1500字）：3-4个工具详细对比
  • 测试体验（500字）：真实使用场景
  • 总结（300字）：选择建议
```

### 如何选择最佳选题

#### 1. 评估维度

```
[AI] 分析选题
对比以下选题：
1. 选题A：[描述]
2. 选题B：[描述]
3. 选题C：[描述]

评估标准：
- 读者兴趣度
- 内容独特性
- 写作难度
- 资源要求
- 爆炸性潜力
```

#### 2. 选择技巧

- **数据类文章**：选择有真实数据的
- **体验类文章**：选择自己真正体验过的
- **趋势类文章**：选择有明确证据的趋势

## ❌ 常见误区

### 1. 避免的选题

- 纯理论分析（缺乏实践）
- 过于宽泛的话题（难以深入）
- 过时的内容（信息陈旧）
- 纯观点输出（缺乏支撑）

### 2. 警惕的问题

- **标题党**：标题吸引但内容空洞
- **同质化**：别人写过的内容
- **无验证**：缺乏数据和事实支撑
- **无价值**：对读者没有实际帮助
