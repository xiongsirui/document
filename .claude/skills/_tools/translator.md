# GLM智能翻译器

基于GLM模型的高质量翻译服务，支持多种语言。

## 翻译功能

### 中英互译
中文与英文之间的双向翻译。

**使用示例：**
```
请用glm_translator把这句话翻译成英文："人工智能正在改变世界"
```

```
请用glm_translator把这句话翻译成中文："Artificial intelligence is changing the world"
```

### 多语言翻译
支持日文、韩文、法文、德文等多种语言。

**使用示例：**
```
请用glm_translator把这段话翻译成日文："你好，世界！"
```

### 专业领域翻译
针对特定领域的专业翻译。

**使用示例：**
```
请用glm_translator翻译这段技术文档，保持专业术语准确："机器学习是人工智能的一个分支..."
```

### 本地化翻译
不仅是翻译，还要符合当地文化习惯。

**使用示例：**
```
请用glm_translator把这个营销文案本地化到美国市场："新品上市，限时优惠！"
```

## 特色功能

### 上下文保持
翻译时保持原文的语境和语气。

### 术语一致性
多次翻译同一术语时保持一致。

### 格式保留
保留原文的格式（如列表、标题等）。

### 批量翻译
支持同时翻译多段内容。

**使用示例：**
```
请用glm_translator批量翻译以下内容：
1. 第一段内容
2. 第二段内容
3. 第三段内容
```

## 使用技巧

1. **提供上下文**：告诉翻译场景，获得更好效果
2. **指定受众**：说明目标读者，调整翻译风格
3. **保持简洁**：避免过长的句子，提高准确性
4. **多次优化**：可以要求重新翻译或调整
