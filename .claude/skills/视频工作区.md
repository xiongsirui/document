# 视频创作工作区

## 核心规则

### 内容要求
- ✅ 开头3秒必须抓住注意力
- ✅ 每分钟至少一个信息点
- ✅ 视觉化呈现优先
- ✅ 需要背景音乐建议

### 脚本结构
```
1. 开头（0-5秒）：悬念/冲突/金句
2. 介绍（5-15秒）：主题介绍
3. 主体（15秒-时长-30秒）：
   - 观点1 + 镜头1
   - 观点2 + 镜头2
   - 观点3 + 镜头3
4. 结尾（最后30秒）：总结 + CTA
```

### 分镜头格式
```
[镜头序号] 时间 | 景别 | 画面内容 | 音效/音乐
例：
[001] 0-3s | 特写 | 产品logo出现 | 科技音效
[002] 3-10s | 中景 | 主持人出场 | 轻快BGM
```

### 时长规范
- **短视频**：15-60秒
- **中等视频**：1-5分钟
- **长视频**：5-15分钟

## 输出格式

```markdown
# 视频脚本：[标题]

## 基本信息
- 时长：X分Y秒
- 风格：[轻松/专业/悬疑]
- 目标：[涨粉/带货/知识]

## 分镜头脚本
[镜头序号] | 时间 | 景别 | 画面 | 音效/字幕

## 拍摄要点
1. [注意事项1]
2. [注意事项2]

## 剪辑建议
- 转场：[类型]
- 字幕：[样式]
- BGM：[风格]
```

## 平台特色
### 抖音/快手
- 竖屏优先
- 强节奏感
- 互动引导

### B站
- 横屏为主
- 深度内容
- 弹幕互动

### 小红书
- 精美画面
- 真人出镜
- 标签优化
