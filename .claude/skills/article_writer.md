# 文章写作助手

提供多种文章类型的写作模板，快速生成高质量内容。

## 使用方法

选择一个模板并提供必要信息。

## 文章模板

### 1. 新闻报道
```
使用模板：新闻报道
主题：[事件主题]
时间：[发生时间]
地点：[发生地点]
关键人物：[相关人物]
事件经过：[详细描述]
影响：[造成的影响]
```

### 2. 产品介绍
```
使用模板：产品介绍
产品名称：[产品名]
产品类型：[软件/硬件/服务]
核心功能：[主要功能1, 功能2, 功能3]
目标用户：[用户群体]
独特优势：[区别于竞品的特点]
使用场景：[具体应用场景]
价格：[价格信息]
```

### 3. 技术教程
```
使用模板：技术教程
技术主题：[技术名称]
难度级别：[初级/中级/高级]

步骤列表：
1. [第一步]
2. [第二步]
3. [第三步]
注意事项：[重要提醒]

```

### 4. 观点评论
```
使用模板：观点评论

论据3：[支持观点的理由]

### 5. 故事创作
```

## 输出示例

**输入：**
```
使用模板：新闻报道
主题：AI绘画工具发布
时间：2024年1月
地点：北京
关键人物：李明（产品经理）
事件经过：推出新的AI绘画工具，支持多种风格
影响：降低创作门槛，推动行业发展
```

**输出：**
```
# 北京发布全新AI绘画工具，让创作更简单

2024年1月，北京 — 今日，科技创新公司在北京正式发布了其最新研发的AI绘画工具"ArtisticAI"。该产品由产品经理李明带领团队开发，标志着AI在创意领域的新突破。

据李明介绍，ArtisticAI采用了最新的深度学习技术，能够支持油画、水彩、素描、动漫等多种艺术风格的自动生成。用户只需输入简单的文字描述，即可在几秒钟内获得高质量的绘画作品。

这款工具的发布预计将对创意产业产生深远影响。首先，它大大降低了艺术创作的门槛，让没有专业绘画技能的用户也能创作出精美的作品。其次，它为专业艺术家提供了强大的辅助工具，能够快速生成创意原型，提高创作效率。

业内专家表示，ArtisticAI的推出不仅展示了AI在创意领域的巨大潜力，也为整个行业的发展指明了方向。随着技术的不断进步，未来AI将在更多创意领域发挥重要作用。
```

## 使用技巧

1. **信息完整**：提供尽可能详细的信息
2. **明确目标**：清楚说明文章的用途和读者
3. **选择合适模板**：根据内容类型选择最匹配的模板
4. **灵活调整**：可以根据需要修改模板结构

## 支持的语言

- 中文（默认）
- 英文（在模板中指定即可）
