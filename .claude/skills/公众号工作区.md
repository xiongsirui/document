# 公众号写作工作区

## 核心规则

### 内容要求
- ✅ 必须有吸引人的标题
- ✅ 需要配图建议（至少3处）
- ✅ 字数控制：1500-3000字
- ✅ SEO优化：关键词布局
- ✅ 结尾要有互动引导

### 结构模板
```
1. 标题（吸引点击）
2. 开头（200字）：痛点/问题/悬念
3. 主体（1500-2500字）：
   - 观点1 + 案例
   - 观点2 + 数据
   - 观点3 + 方法
4. 结尾（200字）：总结 + 行动号召
```

### 配图要求
- **封面图**：标题视觉化
- **内容图**：关键数据图表
- **结尾图**：二维码/关注引导

### 禁忌
- ❌ 纯理论堆砌
- ❌ 没有个人观点
- ❌ 缺少真实案例
- ❌ 标题平淡无奇

## 输出格式

```markdown
# 文章标题

## 开头
[引人入胜的开场]

## 正文
### 小标题1
[内容 + 案例]

### 小标题2
[内容 + 数据]

### 小标题3
[内容 + 方法]

## 总结
[总结 + 行动号召]

---
## 配图建议
1. 封面图：[描述]
2. 内容图1：[位置 + 描述]
3. 结尾图：[二维码位置]
```

## 质量检查清单
- [ ] 标题是否吸引人？
- [ ] 是否有3个以上配图建议？
- [ ] 字数是否在范围内？
- [ ] 是否包含SEO关键词？
- [ ] 结尾是否有互动引导？
