# GLM风格写作助手

模拟GLM模型的写作风格，提供高质量的内容生成服务。

## 写作风格特点

- 简洁明了，直击要点
- 逻辑清晰，结构严谨
- 语言通俗易懂
- 善用实例和案例
- 注重实用性

## 功能

### 文章生成
按照GLM风格生成各类文章。

**使用示例：**
```
请用glm_writer_prompt风格写一篇关于远程工作的优势的文章，要求1000字左右
```

### 内容改写
将内容改写为GLM风格。

**使用示例：**
```
请用glm_writer_prompt风格改写这段话："在当前快速发展的数字时代，我们必须采用创新的方法论来优化我们的工作流程..."
```

### 标题创作
生成简洁有力的标题。

**使用示例：**
```
请用glm_writer_prompt风格为"人工智能在教育中的应用"生成5个标题
```

## 特点说明

- **成本效益**：使用Claude而非GLM API，避免额外费用
- **风格模拟**：通过Prompt工程模拟GLM的写作风格
- **即时可用**：无需配置API Key

## 使用限制

- 是风格模拟，非真正的GLM模型
- 效果取决于Prompt的准确性
- 某些功能可能不如原生GLM强大
