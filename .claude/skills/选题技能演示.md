# Skills 选题功能演示

## 快速选题的三种 Skills 使用方式

### 方式1：使用 writing_workflow Skill（全自动）

直接在对话中输入：
```
[Workflow] 开始写作流程
主题：[你的主题]
类型：公众号
目标字数：2000
```

**系统会自动执行**：
1. Step1 - 创建 Brief（分析需求）
2. Step2 - 信息搜索（收集资料）
3. Step3 - 选题讨论（提供3个选题方案）

### 方式2：知识库 + AI Skill（基于个人素材）

**第一步**：搜索个人素材
```
[KB] 搜索素材
关键词：[主题关键词]
类型：经历, 评价, 案例
```

**第二步**：基于素材生成选题
```
[AI] 生成选题
素材：[选择找到的素材]
要求：公众号，2000字
```

### 方式3：使用 writer_api Skill（快速生成）

```
[AI] 生成选题
主题：[你的主题]
数量：5个
风格：[吸引点击/专业深度/实用干货]
平台：公众号
```

## 实际案例演示

### 案例：想写"AI效率工具"相关文章

#### 使用方式1：
```
用户: [Workflow] 开始写作流程
主题：AI效率工具
类型：公众号
目标字数：2500

系统执行：
✓ Step1 - 分析目标读者（职场人士）
✓ Step2 - 搜索2025年最新AI工具趋势
✓ Step3 - 提供3个选题：
  1. 《2025年最值得关注的5款AI效率工具深度评测》
  2. 《从入门到精通：我的AI效率工具使用实践》
  3. 《AI工具如何重塑知识管理工作流》
```

#### 使用方式2：
```
用户: [KB] 搜索素材
关键词：AI工具, 效率, 测试
类型：经历, 评价

系统找到：
- 测试过Notion AI的体验
- ChatGPT插件使用心得
- 自动化工具实践记录

用户: [AI] 生成选题
素材：上述素材
要求：公众号，2000字

系统生成：
基于你的真实体验，推荐以下选题...
```

## Skills 选题的优势

1. **智能化**：自动搜索最新信息
2. **个性化**：结合个人素材库
3. **多样化**：提供多个角度的选题
4. **专业化**：符合公众号传播特点

## 技巧提示

- 题目越具体，选题越精准
- 多使用个人素材，内容更真实
- 可以结合多个 Skills 使用
- 选题后可以直接继续写作流程
