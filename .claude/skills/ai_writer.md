# AI 写作助手

直接使用 Claude 的能力进行创作，无需启动后端服务。

## 功能特点

- ✅ 无需 Python 后端
- ✅ 直接使用 Claude Code
- ✅ 支持保存到指定路径
- ✅ 内置多种写作模板

## 使用方法

### 1. 写文章
```
[AI] 写文章
主题：[文章主题]
风格：[专业/轻松/学术/营销]
字数：[期望字数]
保存到：[文件路径，可选]
```

### 2. 生成选题
```
[AI] 生成选题
主题：[主题领域]
数量：[生成数量，默认3个]
风格：[吸引点击/专业深度/实用干货]
```

### 3. 保存文档
```
[AI] 保存文章
内容：[文章内容]
文件名：[文件名.md]
路径：[保存路径，默认当前目录]
```

### 4. 优化文本
```
[AI] 优化文本
内容：[粘贴文本]
目标：[更简洁/更专业/更生动/更易读]
```

### 5. 创建大纲
```
[AI] 创建大纲
主题：[文章主题]
字数：[预计字数]
角度：[切入点/观点]
```

## 示例

### 示例1：写一篇技术文章
```
[AI] 写文章
主题：Claude Code 使用技巧
风格：轻松
字数：1500
保存到：articles/claude-tips.md
```

### 示例2：生成公众号选题
```
[AI] 生成选题
主题：AI工具
数量：5
风格：实用干货
```

### 示例3：优化文章风格
```
[AI] 优化文本
内容：[粘贴你的文章]
目标：更轻松活泼
```

## 保存路径说明

- 默认保存在当前工作目录（`d:\code\ai\document\`）
- 可以使用相对路径：`output/2025/12/`
- 也可以使用绝对路径：`D:\MyDocs\`

## 注意事项

1. 此 Skill 直接使用 Claude 的能力，无需启动 glm_backend.py
2. 文件会自动创建 `.md` 后缀
3. 路径不存在时会自动创建
4. 支持中文路径和文件名
