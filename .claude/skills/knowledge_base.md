# 个人知识库

管理和搜索个人经历、评价、案例等真实素材，用于写作时的素材引用。

## 核心功能

### 1. 添加素材

```
[KB] 添加素材
类型：[经历/评价/案例/洞察/数据]
内容：[素材内容]
标签：[标签1, 标签2, 标签3]
日期：[YYYY-MM-DD]
来源：[即刻/微博/笔记/实践]
可信度：[1-10分]
```

**示例：**
```
[KB] 添加素材
类型：产品评价
内容：今天测试了Claude Code两周，发现它在代码生成方面确实比Cursor更准确。特别是Python代码，几乎不需要修改。但UI交互还不够流畅。
标签：Claude Code, 测试评价, 编程工具
日期：2024-12-15
来源：个人测试
可信度：9
```

### 2. 搜索素材

```
[KB] 搜索素材
关键词：[关键词1, 关键词2]
类型：[经历/评价/案例]
时间范围：[最近7天/最近30天/最近一年]
排序：[时间/相关度/可信度]
```

**示例：**
```
[KB] 搜索素材
关键词：AI编程工具, 评价
类型：评价
时间范围：最近30天
排序：相关度
```

### 3. 查看主题索引

```
[KB] 主题索引
主题：[主题名称]
```

会显示该主题下的所有素材分类。

### 4. 生成素材报告

```
[KB] 生成报告
主题：[主题名称]
时间范围：[自定义/默认全部]
```

## 搜索技巧

1. **组合搜索**：使用多个相关关键词
2. **时间筛选**：优先使用最近的素材
3. **可信度过滤**：高可信度素材优先
4. **场景匹配**：根据文章部分选择合适的素材类型

## 注意事项

- 所有素材必须是真实的，不能编造
- 保护隐私，不要记录敏感信息
- 定期清理和更新旧素材
- 保持标签的一致性和准确性
