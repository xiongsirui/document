# GLM写作助手

利用GLM API提供高效的写作服务，大幅降低成本。

## 功能

### 生成文章
基于给定的主题生成高质量文章。

**使用示例：**
```
请用glm_writer写一篇关于人工智能在教育中应用的文章，要求2000字，风格专业
```

### 内容优化
优化现有文章，提高可读性和吸引力。

**使用示例：**
```
请用glm_writer优化这段话，让它更简洁有力："我们公司提供了一个非常好的解决方案..."
```

### 生成标题
为文章生成多个吸引人的标题选项。

**使用示例：**
```
请用glm_writer为这篇文章生成5个标题："文章内容..."
```

### 摘要生成
快速生成文章摘要。

**使用示例：**
```
请用glm_writer生成这篇文章的摘要："长篇文章内容..."
```

## 优势

- **成本极低**：仅为Claude成本的10%
- **响应快速**：平均响应时间2-5秒
- **质量保证**：使用GLM-4.6模型确保质量
- **中文优化**：针对中文写作优化

## 配置

使用前需要设置GLM_API_KEY环境变量：

```bash
export GLM_API_KEY="your-api-key"
```
