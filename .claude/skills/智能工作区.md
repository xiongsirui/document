# 智能工作区系统

## 快速开始

直接说你的需求，系统会自动识别：

```
[开始] 写一篇关于AI的文章
[开始] 做个产品介绍视频
[开始] 优化这个提示词
[开始] 帮我分析个问题
```

## 自动识别逻辑

### 公众号识别
- 关键词：公众号、文章、写作、内容、推文
- 自动调用：`writing_workflow.md`
- 应用规则：`公众号工作区.md`

### 视频识别
- 关键词：视频、脚本、剪辑、短视频、Vlog
- 自动调用：`workflow_lite.md`（视频模式）
- 应用规则：`视频工作区.md`

### Prompt识别
- 关键词：prompt、提示词、AI指令、优化prompt
- 自动调用：`Prompt工作区.md`
- 输出：标准化Prompt模板

### 通用任务
- 其他所有任务
- 灵活处理
- 按需定制

## 手动指定模式

当自动识别不准确时，可以手动指定：

```
[公众号] 不管输入什么，都用公众号工作区
[视频] 强制使用视频工作区
[Prompt] 强制使用Prompt工作区
[通用] 使用通用任务模式
```

## 工作区切换

任务进行中可以切换：

```
[切换视频] 把这篇文章改成视频脚本
[切换Prompt] 总结成prompt模板
[切换公众号] 扩展成公众号文章
```

## 组合任务

支持复杂组合任务：

```
[组合] 先写公众号文章，再改编成视频脚本
```

执行流程：
1. 先执行公众号工作区
2. 完成后自动切换到视频工作区
3. 基于文章内容生成脚本

## 智能建议

当任务不明确时：

```
用户：帮我做内容

系统：请选择内容类型：
1. 📝 公众号文章
2. 🎬 视频脚本
3. 💡 Prompt优化
4. 📋 其他（请说明）

您可以回复数字或直接描述需求
```

## 实际演示

### 复杂任务处理
```
用户：我想做个AI教程，既要文章又要视频

系统：识别到组合任务
[计划] 将分两步执行：
1. 先创作公众号文章（详细教程）
2. 再改编成视频脚本（精简版）

是否开始？[Y/n]
```

### 模糊任务处理
```
用户：AI帮我工作

系统：任务不够明确，请补充：
- 是写AI使用心得？(公众号)
- 还是做AI演示视频？(视频)
- 或是优化AI的prompt？(Prompt)
```

## 配置自定义

可以自定义工作区规则：
- 编辑 `公众号工作区.md` 调整写作规则
- 编辑 `视频工作区.md` 修改脚本格式
- 添加新的工作区类型

## 工作区扩展

支持添加新的工作区：
```
[新工作区] 播客工作区
[新工作区] 课程工作区
[新工作区] 营销文案工作区
```

开始使用：直接说"[开始] 你的任务"即可！
