# 轻量版写作工作流

实现核心写作流程，无需启动后端服务，直接使用 Claude 的能力。

## 快速启动

```
[Workflow] 开始写作流程
主题：[文章主题]
类型：[公众号/博客/报告]
目标字数：[字数]
```

## 完整流程（可选步骤）

### Step 1: 需求分析
```
[Workflow] Step1 - 需求分析
```
系统会分析：
- 目标读者
- 内容定位
- 核心要点
- 传播价值

### Step 2: 选题方案
```
[Workflow] Step2 - 生成选题
```
系统提供3个选题：
- 选题1：工具评测类
- 选题2：经验方法类
- 选题3：趋势分析类

### Step 3: 创建大纲
```
[Workflow] Step3 - 创建大纲
选题：[选择哪个选题]
```
生成详细大纲，包括：
- 开头（吸引读者）
- 主体（3-5个要点）
- 结尾（总结升华）

### Step 4: 创作初稿
```
[Workflow] Step4 - 创作初稿
大纲：[基于之前的大纲]
风格：[专业/轻松/学术]
```

### Step 5: 优化完善
```
[Workflow] Step5 - 优化文章
优化重点：[开头吸引力/案例丰富度/结尾行动号召]
```

## 示例演示

### 完整示例
```
用户：[Workflow] 开始写作流程
主题：AI编程工具
类型：公众号
目标字数：2500

系统执行 Step1-Step5：
✓ 分析需求：面向开发者的工具对比
✓ 提供3个选题
✓ 生成详细大纲
✓ 创作初稿
✓ 优化完善
```

### 分步执行
```
# 只执行选题
用户：[Workflow] Step2 - 生成选题
主题：远程办公效率

系统直接提供3个选题方案
```

## 保存功能

```
[Workflow] 保存文章
文件名：文章标题.md
路径：output/2025/12/
```

## 与完整版的区别

| 功能 | 轻量版 | 完整版 |
|------|--------|--------|
| 后端服务 | ❌ 不需要 | ✅ 需要 |
| 自动搜索 | ❌ 需手动提供 | ✅ 自动搜索 |
| 知识库 | ❌ 不支持 | ✅ 支持 |
| 步骤数 | 5步核心 | 9步完整 |
| 成本 | Claude tokens | GLM API（更便宜） |
| 响应速度 | 快 | 慢（多步骤） |

## 使用建议

1. **日常写作**：使用轻量版，足够快速生成高质量内容
2. **快速出稿**：直接跳到 Step4 创作初稿
3. **仅要选题**：只执行 Step2
4. **正式发布**：如果追求极致质量，可考虑完整版

## 注意事项

- 轻量版可以获取最新信息（Claude有网络搜索能力）
- 如需特定最新资料，可以明确要求："请搜索XX的最新信息"
- 可以随时中断流程，使用 `[Workflow] 跳到 Step4]`
