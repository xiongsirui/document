# GLM知识管理助手

智能的知识管理工具，帮助组织、检索和扩展知识库。

## 核心功能

### 知识提取
从文本中自动提取和结构化知识。

**使用示例：**
```
请用glm_knowledge从以下文档中提取关键知识点：
"机器学习是人工智能的一个分支，它使计算机能够在没有明确编程的情况下学习..."
```

### 概念解释
提供专业概念和术语的详细解释。

**使用示例：**
```
请用glm_knowledge解释什么是"区块链"，包括原理和应用
```

### 知识关联
发现不同知识点之间的联系。

**使用示例：**
```
请用glm_knowledge分析"深度学习"和"神经网络"之间的关系
```

### 学习路径
为特定主题创建学习计划。

**使用示例：**
```
请用glm_knowledge为"前端开发"创建一个3个月的学习路径
```


## 使用技巧

1. **明确目标**：说明学习目的和应用场景
2. **提供背景**：分享已知信息，获得更精准的帮助
